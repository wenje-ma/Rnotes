# R：08列表
列表是 R 语言**最灵活、最强大**的数据结构。

特点：

- 可以存放**任意类型**的数据（数值、字符串、向量、矩阵、甚至列表）
- 长度可变
- 可命名、可索引、可遍历

## 一、创建列表

使用 `list()` 创建列表。

```r
# 创建一个混合类型的列表
list1 <- list("runoob", "google", c(11,22,33), 123, 51.23)
print(list1)
```

### 给列表元素命名

命名后可以用 **$名字** 访问元素。

```r
list_data <- list(
  c("Google","Runoob","Taobao"),
  matrix(c(1,2,3,4,5,6),2,3),
  list("a",12,3)
)

# 命名
names(list_data) <- c("Sites", "Numbers", "Lists")

print(list_data)
```

## 二、访问列表元素

两种方式：

1. **按索引**：`list[1]`
2. **按名称**：`list$名字`

```r
# 按索引
print(list_data[1])

# 按名称（最常用）
print(list_data$Sites)
print(list_data$Numbers)
```

## 三、列表元素操作

### 1. 添加元素

直接给新位置赋值即可。

```r
list_data[4] <- "新元素"
```

### 2. 删除元素

赋值为 `NULL`。

```r
list_data[4] <- NULL
```

### 3. 修改元素

直接覆盖。

```r
list_data$Sites <- "新网站"
```

## 四、列表循环遍历

### 1. for 循环遍历列表

```r
my_list <- list(1,2,3,"a","b","c")
for (elem in my_list) {
  print(elem)
}
```

### 2. lapply 函数

对列表**每个元素执行函数**，返回**新列表**。

```r
my_list <- list(1,2,3)
result <- lapply(my_list, function(x) x * 2)
print(result)
```

## 五、合并列表

### 1. 使用 c() 合并多个列表

```r
list1 <- list(1,2,3)
list2 <- list("a","b","c")
combined <- c(list1, list2)
print(combined)
```

### 2. 使用 append() 指定位置插入

```r
# 在第 2 个元素后插入
list_new <- append(list1, list2, after = 2)
print(list_new)
```

## 六、列表 → 向量

列表无法直接计算，必须转成向量。

```r
my_list <- list(1:5)
v <- unlist(my_list)
print(v)
```

## 七、列表核心函数速查表

| 函数 | 作用 |
|------|------|
| `list()` | 创建列表 |
| `names()` | 给列表命名 |
| `list$name` | 按名称访问元素 |
| `list[i]` | 按索引访问 |
| `lapply()` | 对列表每个元素应用函数 |
| `c(list1, list2)` | 合并列表 |
| `append()` | 指定位置插入 |
| `unlist()` | 列表转向量 |
