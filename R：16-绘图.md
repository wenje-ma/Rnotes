## 一、饼图 pie()

用于展示**各部分占整体的比例**，适合分类占比数据。

### 核心语法

```r
pie(x, labels = 名称, col = 颜色, main = 标题)
legend(位置, 图例, fill = 颜色)  # 添加图例
```

### 完整示例

```r
# 1. 构造数据
info <- c(1, 2, 4, 8)
site_names <- c("Google", "Runoob", "Taobao", "Weibo")

# 2. 计算百分比标签
piepercent <- paste(round(100 * info / sum(info)), "%")

# 3. 绘制饼图
pie(info, 
    labels = piepercent,  # 扇形标签
    main = "网站访问占比", # 标题
    col = 1:4)            # 颜色

# 4. 添加右上角图例
legend("topright", site_names, fill = 1:4)
```

### 关键参数

- `x`：数值向量，决定扇形大小
- `labels`：每个扇形的文字标签
- `col`：填充颜色（`1:4` 调用默认调色板前4色）
- `legend()`：专门为饼图添加图例，`fill` 匹配饼图颜色

## 二、条形图 barplot()

用于展示**分类数据的大小对比**，支持单组/多组数据。

### 核心语法

```r
barplot(height, names.arg = 分类名, beside = FALSE, col = 颜色, main = 标题, legend = FALSE)
```

### 1. 单组数据条形图

```r
# 数据
cvd19 <- c(83534, 2640626, 585493)

# 绘图
barplot(cvd19,
        main = "新冠疫情累计确诊",
        col = 1:3,
        names.arg = c("中国", "美国", "印度")) # 分类标签
```

### 2. 多组数据：矩阵+并列/堆叠

矩阵格式：**行=分组，列=分类**

- `beside = TRUE`：**并列条形图**
- `beside = FALSE`（默认）：**堆叠条形图**

```r
# 构造矩阵数据（2行3列）
cvd19 <- matrix(c(83017, 83534, 1794546, 2640626, 190535, 585493),
                nrow = 2, ncol = 3)

# 设置行列名
colnames(cvd19) <- c("中国", "美国", "印度")
rownames(cvd19) <- c("6月", "7月")

# 并列条形图（推荐对比用）
barplot(cvd19,
        main = "6-7月疫情对比",
        beside = TRUE,  # 并列
        legend = TRUE)  # 自动生成图例

# 堆叠条形图
barplot(cvd19, main = "6-7月疫情堆叠", legend = TRUE)
```

### 关键参数

- `height`：向量/矩阵，条形高度
- `beside`：`TRUE`并列，`FALSE`堆叠
- `legend`：`TRUE`自动生成图例
- `names.arg`：横轴分类名称

## 三、函数曲线图 curve()

快速绘制**数学函数图像**，无需手动生成x/y数据。

### 核心语法

```r
curve(函数表达式, from = 起始值, to = 终止值)
```

### 示例

```r
# 绘制正弦函数，范围：-2π ~ 2π
curve(sin(x), from = -2*pi, to = 2*pi, 
      main = "正弦函数曲线", 
      col = "blue", lwd = 2)
```

## 四、散点图 plot()

展示**两个变量之间的关系**，是最常用的基础图表。

### 核心语法

```r
plot(x, y, type = "p", main = 标题, xlab = X轴名, ylab = Y轴名)
```

### type 参数

| 参数 | 样式 |
|------|------|
| p | 点图（默认） |
| l | 线图 |
| b | 点+线 |
| o | 线穿过点 |
| h | 竖线图 |

### 示例

```r
# 生成数据
x <- seq(-2, 2, length = 100)
y <- sin(x)

# 绘制线图
plot(x, y, 
     type = "l",    # 线图样式
     main = "正弦函数散点线图",
     xlab = "X值",
     ylab = "sin(X)",
     col = "red", lwd = 2)
```

## 五、散点图矩阵 pairs()

一次性展示**多个变量两两之间的相关性**，适合多变量分析。

### 核心语法

```r
pairs(~ 变量1 + 变量2 + 变量3, data = 数据集)
```

### 示例（年龄、身高、体重相关性）

```r
set.seed(123)  # 固定随机种子，结果可复现
n <- 100

# 生成模拟数据
age <- round(runif(n, 18, 60))    # 年龄18-60岁
height <- round(rnorm(n, 170, 8)) # 身高均值170
weight <- round(rnorm(n, 65, 10) + height*0.3) # 体重

# 构建数据框
data <- data.frame(age, height, weight)

# 绘制散点图矩阵
pairs(~ age + height + weight, data = data, main = "多变量相关性")
```

### 图表含义
- 对角线：变量名称
- 非对角线：两个变量的散点图，直观展示相关性
