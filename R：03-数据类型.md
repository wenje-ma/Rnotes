# R：03-数据类型

## 一、R 基础数据类型

R 有 **3 种最基本数据类型**：

1. **数字型**（numeric）
2. **逻辑型**（logical）
3. **文本型**（character）

### 1. 数字常量

- 一般型：`123`、`-0.125`
- 科学计数法：`1.23e2`、`-1.25E-1`

### 2. 逻辑型（Bool）

只有两个值：

- `TRUE`
- `FALSE`

⚠️ **R 对大小写敏感**，`true`、`false` 不是合法布尔值。

### 3. 文本型（字符串）

用 **双引号 `"`** 包裹。

```r
print("runoob" == "runoob")
```

```
[1] TRUE
```

## 二、R 6 大对象类型

R 变量都是**对象**，常用 6 种：

1. 向量（Vector）
2. 列表（List）
3. 矩阵（Matrix）
4. 数组（Array）
5. 因子（Factor）
6. 数据框（Data Frame）

## 三、向量

向量是**相同类型元素的序列**，用 `c()` 创建。

### 1. 创建与运算

```r
a <- c(3, 4)
b <- c(5, 0)
print(a + b)
```

```
[1] 8 4
```

### 2. 访问元素（下标从 1 开始）

```r
a <- c(10, 20, 30, 40, 50)
print(a[2])
```

```
[1] 20
```

### 3. 截取子集

```r
a <- c(10,20,30,40,50)
print(a[1:4])       # 1~4 项
print(a[c(1,3,5)])  # 第 1、3、5 项
print(a[c(-1,-5)])  # 去掉 1、5 项
```

### 4. 标量运算（向量化运算）

每个元素同时计算：

```r
print(c(1.1,1.2,1.3) - 0.5)
a <- c(1,2)
print(a^2)
```

### 5. 排序函数

```r
a <- c(1,3,5,2,4,6)
print(sort(a))    # 从小到大排序
print(rev(a))     # 反转顺序
print(order(a))   # 返回排序后的下标
print(a[order(a)])# 等价 sort(a)
```

### 6. 向量统计函数

| 函数 | 功能 |
|||
| `sum` | 求和 |
| `mean` | 均值 |
| `var` | 方差 |
| `sd` | 标准差 |
| `min` | 最小值 |
| `max` | 最大值 |
| `range` | 最小、最大值 |

```r
print(sum(1:5))
print(sd(1:5))
print(range(1:5))
```

### 7. 向量生成

- 连续序列：`1:10`
- 等差数列：`seq(首, 末, 步长)`
- 重复序列：`rep(值, 次数)`

```r
print(seq(1,9,2))
print(seq(0,1,length.out=3))
print(rep(0,5))
```

### 8. NA 与 NULL

- `NA`：数据**缺失**，位置保留
- `NULL`：数据**不存在**，无位置

```r
print(length(c(NA,NA,NULL)))
print(c(NA,NA,NULL,NA))
```

## 四、逻辑型向量

用于条件判断与筛选。

### 1. 生成逻辑向量

```r
print(c(11,12,13) > 12)
```

```
[1] FALSE FALSE TRUE
```

### 2. which()：返回 TRUE 位置

```r
a <- c(11,12,13)
b <- a > 11
print(which(b))
```

```
[1] 2 3
```

### 3. 数据筛选示例

```r
vec <- c(10,40,78,64,53,62,69,70)
print(vec[which(vec >= 60 & vec < 70)])
```

```
[1] 64 62 69
```

### 4. all() 与 any()

```r
print(all(c(T,T,T)))  # 全部为 TRUE？
print(any(c(T,F,F)))  # 至少一个 TRUE？
```

## 五、字符串操作

常用字符串处理函数：

```r
print(toupper("Runoob"))       # 大写
print(tolower("Runoob"))       # 小写
print(nchar("中文",type="char"))# 字符数
print(substr("123456789",1,5)) # 截取
print(as.numeric("12"))        # 转数字
print(as.character(12.34))     # 转字符串
print(strsplit("2019;10;1",";"))# 拆分
print(gsub("/","-","2019/10/1"))# 替换
```

### 正则替换

```r
print(gsub("[0-9]","X","abc123"))
print(gsub("\\d","X","abc123",perl=TRUE))
```

## 六、矩阵

矩阵是**二维、同类型**数据结构。

### 1. 创建矩阵

```r
vec <- c(1,2,3,4,5,6)
print(matrix(vec, 2, 3))        # 默认按列填充
print(matrix(vec, 2, 3, byrow=T))# 按行填充
```

### 2. 访问元素

```r
m1 <- matrix(vec,2,3,byrow=T)
print(m1[1, 3]) # 第1行第3列
```

### 3. 行名、列名

```r
colnames(m1) <- c("x","y","z")
rownames(m1) <- c("a","b")
print(m1["a", ]) # 取 a 行
```

### 4. 矩阵运算

- 数乘：`*`
- **矩阵乘法**：`%*%`
- 逆矩阵：`solve()`

```r
m1 <- matrix(c(1,2),1,2)
m2 <- matrix(c(3,4),2,1)
print(m1 %*% m2)
```

### 5. 逆矩阵

```r
m <- matrix(c(1,3,2,4), 2, 2)
print(solve(m))
```

### 6. 按行/列计算 apply()

```r
print(apply(m, 1, sum)) # 1=行
print(apply(m, 2, sum)) # 2=列
```
