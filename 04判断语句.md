# R：04判断语句

R 语言提供 **3 种核心判断语句**，用于根据条件执行不同代码逻辑：

1. `if` 单条件判断
2. `if...else` / `if...else if...else` 多条件判断
3. `switch` 多值匹配判断

## 一、if 语句（单条件）

### 1. 语法

```r
if (条件表达式) {
  条件为 TRUE 时执行的代码
}
```

### 2. 说明

- 条件必须返回 **TRUE / FALSE**
- 条件为 `TRUE` 执行大括号内代码
- 条件为 `FALSE` 跳过不执行

### 3. 示例

```r
x <- 50L  # L 表示整数类型
if (is.integer(x)) {
  print("x 是整数")
}
```

**输出**：

```
[1] "x 是整数"
```

## 二、if...else 语句

### 1. 语法

```r
if (条件) {
  条件为 TRUE 执行
} else {
  条件为 FALSE 执行
}
```

### 2. 示例（判断元素是否在向量中）

```r
x <- c("google", "runoob", "taobao")
if ("runoob" %in% x) {
  print("包含 runoob")
} else {
  print("不包含 runoob")
}
```

**输出**：

```
[1] "包含 runoob"
```

## 三、if...else if...else 语句（多条件）

### 1. 语法

```r
if (条件1) {
  条件1 为真执行
} else if (条件2) {
  条件2 为真执行
} else if (条件3) {
  条件3 为真执行
} else {
  所有条件都不满足执行
}
```

### 2. 执行规则

- **从上到下依次判断**
- 只要**第一个满足条件**的分支执行，后面全部跳过

### 3. 示例

```r
x <- c("google", "runoob", "taobao")

if ("weibo" %in% x) {
  print("找到 weibo")
} else if ("runoob" %in% x) {
  print("找到 runoob")
} else {
  print("未找到目标")
}
```

**输出**：

```
[1] "找到 runoob"
```

## 四、switch 语句（多值匹配）

`switch` 专门用于**一个变量匹配多个固定值**，比多层 if 更简洁。

### 1. 两种使用方式

#### 方式 1：按**数字位置**匹配

```r
switch(数字, 选项1, 选项2, 选项3, ...)
```

- 数字 = 1 → 返回第 1 个选项
- 数字 = 2 → 返回第 2 个选项
- 数字超出范围 → 返回 `NULL`

**示例**：

```r
res <- switch(3, "google", "runoob", "taobao", "weibo")
print(res)
```

**输出**：

```
[1] "taobao"
```

#### 方式 2：按**字符串名称**匹配

```r
switch(字符串,
       名称1 = 值1,
       名称2 = 值2,
       名称3 = 值3)
```

**示例**：

```r
like <- "runoob"
res <- switch(like,
             google = "www.google.com",
             runoob = "www.runoob.com",
             taobao = "www.taobao.com")
print(res)
```

**输出**：

```
[1] "www.runoob.com"
```

### 2. switch 核心规则

1. **数字匹配**：返回对应位置的值，无匹配返回 `NULL`
2. **字符串匹配**：必须完全匹配名称，无匹配报错
3. 最多**只能有一个默认值**
4. 多个匹配 → 返回**第一个**

## 五、快速总结表

| 语句 | 用途 | 特点 |
|------|------|------|
| `if` | 单条件判断 | 满足才执行 |
| `if...else` | 二选一判断 | 一定执行其中一个 |
| `if...else if...else` | 多条件分支 | 只执行第一个满足的 |
| `switch` | 多值精准匹配 | 数字/字符串匹配，代码更简洁 |

## 六、最常用小技巧

1. 判断元素是否在向量中：`%in%`
   ```r
   "a" %in% c("a","b","c")  # TRUE
   ```
2. R 中数字默认是 `numeric`，加 `L` 才是整数：`5L`
3. `switch(3, ...)` 从 **1 开始计数**，不是 0
