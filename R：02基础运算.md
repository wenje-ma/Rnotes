## 二、数学运算符

优先级数字**越小越先计算**。

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | `()` | 括号 |
| 2 | `^` | 乘方 |
| 3 | `%%` | 取余 |
| 3 | `%/%` | 整除 |
| 4 | `*` `/` | 乘法、除法 |
| 5 | `+` `-` | 加法、减法 |

示例：

```r
print(1 + 2 * 3)        # 7
print((1 + 2) * 3)      # 9
print(3 / 4)            # 0.75
print(3.4 - 1.2)        # 2.2
print(1 - 4 * 0.5^3)    # 0.5
print(8 / 3 %% 2)       # 0.666...
print(8 / 4 %% 2)       # 0
print(3 %% 2^2)         # 3
print(10 / 3 %/% 2)     # 1
```

## 三、关系运算符

用于比较两个向量，返回 **布尔值（TRUE / FALSE）**。

| 运算符 | 说明 |
|--------|------|
| `>` | 大于 |
| `<` | 小于 |
| `==` | 等于 |
| `!=` | 不等于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

示例：

```r
v <- c(2, 4, 6, 9)
t <- c(1, 4, 7, 9)

print(v > t)
print(v < t)
print(v == t)
print(v != t)
print(v >= t)
print(v <= t)
```

## 四、逻辑运算符

对布尔值进行逻辑运算。

**规则**：`0 = FALSE`，非 0 = `TRUE`。

| 运算符 | 说明 |
|--------|------|
| `&` | 元素-wise 与 |
| `|` | 元素-wise 或 |
| `!` | 非 |
| `&&` | 标量 与（只看第一个元素） |
| `||` | 标量 或（只看第一个元素） |

示例：

```r
s <- c(3, 0, TRUE, 2+3i)
t <- c(4, 1, FALSE, 2+3i)

print(s & t)
print(s | t)
print(!s)

u <- 0
v <- 3
print(u && v)
print(u || v)
```

## 五、赋值运算符

| 运算符 | 说明 |
|--------|------|
| `<-` | 局部赋值 |
| `<<-` | 全局赋值 |

```r
v1 <- c(3, 1, TRUE, "runoob")
v2 <<- c(3, 1, TRUE, "runoob")

print(v1)
print(v2)
```

## 六、其他常用运算符

| 运算符 | 用途 |
|--------|------|
| `:` | 生成连续数字向量 |
| `%in%` | 判断元素是否在向量中 |
| `%*%` | 矩阵乘法 |

### 1. 生成连续向量

```r
v <- 1:10
print(v)
```

### 2. 成员判断

```r
v1 <- 3
v2 <- 15
print(v1 %in% v)
print(v2 %in% v)
```

### 3. 矩阵乘法

```r
m <- matrix(c(2,6,5,1,10,4), nrow=2, ncol=3, byrow=TRUE)
print(m)

t <- m %*% t(m)  # t() 转置
print(t)
```

**矩阵参数说明**

- `nrow`：行数
- `ncol`：列数
- `byrow=TRUE`：按行填充
- `t()`：矩阵转置

## 七、数学函数

### 1. 基础数学函数

| 函数 | 说明 |
|------|------|
| `sqrt(n)` | 平方根 |
| `exp(n)` | e 的 n 次方 |
| `log(m,n)` | 以 n 为底 m 的对数 |
| `log10(m)` | 常用对数 |

```r
print(sqrt(4))
print(exp(1))
print(exp(2))
print(log(2,4))
print(log10(10000))
```

### 2. 取整函数

| 函数 | 说明 |
|------|------|
| `round(n)` | 四舍五入 |
| `round(n,m)` | 保留 m 位小数 |
| `ceiling(n)` | 向上取整 |
| `floor(n)` | 向下取整 |

```r
print(round(2.5))
print(round(2.55))
print(ceiling(2.5))
print(floor(2.5))
```

### 3. 三角函数（弧度制）

```r
print(sin(pi/6))
print(cos(pi/4))
print(tan(pi/3))
```

### 4. 反三角函数

```r
print(asin(0.5))
print(acos(0.7071068))
print(atan(1.732051))
```

## 八、统计函数（正态分布）

R 统计函数 = **前缀 + 分布名**

### 前缀含义

- `d`：密度函数
- `p`：分布函数（概率）
- `q`：分位数
- `r`：随机数

### 正态分布：`norm`

```r
print(dnorm(0))     # 标准正态密度
print(pnorm(0))     # 分布函数 P(X≤0)
print(qnorm(0.95))  # 0.95 分位数
```
