# R：11-因子

因子是 R 语言中**专门存储分类数据**的数据类型，核心用于分组、统计、建模等场景，是数据分析的核心数据结构之一。

## 一、因子基础

### 1. 什么是因子？

用于存储**分类/类别数据**（如性别、水果种类、等级），内部以整数编码存储，展示为字符标签。

### 2. 创建因子：`factor()` 函数

#### 标准语法

```r
factor(x, levels, labels = levels, ordered = FALSE)
```

| 参数 | 含义 |
|------|------|
| `x` | 原始向量（字符/数值） |
| `levels` | **手动指定因子的所有类别** |
| `labels` | 给类别设置**别名/标签** |
| `ordered` | 是否为**有序因子**（`TRUE`=有序，`FALSE`=无序） |

#### 基础示例

```r
# 创建水果因子
fruit <- factor(c("苹果", "橘子", "苹果", "苹果", "橘子"))

# 查看因子
print(fruit)
# 判断是否为因子
print(is.factor(fruit))
```

**输出结果**

```
[1] 苹果 橘子 苹果 苹果 橘子
Levels: 苹果 橘子
[1] TRUE
```

## 二、核心用法 1：指定因子类别（levels）

`levels` 可以**限定因子的可选类别**，不在 levels 中的值会自动变为 `NA`（缺失值）。

```r
# 完整指定两个类别
fruit1 <- factor(c("苹果", "橘子", "苹果"), levels = c("苹果", "橘子"))
print(fruit1)

# 仅指定一个类别，橘子会变成 NA
fruit2 <- factor(c("苹果", "橘子", "苹果"), levels = "苹果")
print(fruit2)
```

**输出结果**

```
[1] 苹果 橘子 苹果
Levels: 苹果 橘子

[1] 苹果 <NA> 苹果
Levels: 苹果
```

## 三、核心用法 2：设置类别别名（labels）

将原始编码（如 1/2）替换为可读标签（如 苹果/橘子）。

```r
# 原始数据：1,2 编码
code <- c(1, 2, 1, 1, 2)

# 转换为带标签的因子
fruit <- factor(
  x = code,
  levels = c(1, 2),   # 原始类别
  labels = c("苹果", "橘子")  # 替换后的标签
)

print(fruit)
```

**输出结果**
```
[1] 苹果 橘子 苹果 苹果 橘子
Levels: 苹果 橘子
```

## 四、核心用法 3：有序因子（ordered）

`ordered=TRUE` 表示**类别有大小/顺序关系**，输出会显示 `<` 表示顺序。

```r
fruit <- factor(
  c(1, 2, 1, 1, 2),
  levels = c(1, 2),
  labels = c("苹果", "橘子"),
  ordered = TRUE  # 开启有序
)

print(fruit)
```

**输出结果**

```
[1] 苹果 橘子 苹果 苹果 橘子
Levels: 苹果 < 橘子
```

## 五、快速生成因子：`gl()` 函数

`gl()` = generate levels，**直接生成规则化的因子**，无需原始数据。

### 语法

```r
gl(n, k, length, labels, ordered = FALSE)
```

| 参数 | 含义 |
|------|------|
| `n` | 类别总数 |
| `k` | 每个类别重复次数 |
| `length` | 总长度（可选） |
| `labels` | 类别名称 |
| `ordered` | 是否有序 |

### 示例

```r
# 3个类别，每个重复4次，有序因子
v <- gl(3, 4, labels = c("苹果", "梨", "橘子"), ordered = TRUE)
print(v)
```

**输出结果**

```
 [1] 苹果 苹果 苹果 苹果 梨   梨   梨   梨   橘子 橘子 橘子 橘子
Levels: 苹果 < 梨 < 橘子
```

## 六、函数总结表

| 函数 | 用途 | 核心参数 |
|------|------|----------|
| `factor()` | 将向量转为因子 | `levels`/`labels`/`ordered` |
| `gl()` | 直接生成规则因子 | `n`/`k`/`labels` |
| `is.factor()` | 判断是否为因子 | - |
