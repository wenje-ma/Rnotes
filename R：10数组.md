# R：10数组

数组是同一类型元素的集合，向量是1维数组，矩阵是2维数组。

创建数组使用`array()`函数。

语法格式：

```r
array(data = NA, dim = length(data), dimnames = NULL)
```

- `data`：数组元素，会依据维数循环填满数组；
- `dim`：数组的维数；
- `dimnames`：每个维度的名称。

R中数组索引从1开始。

列表实际上是更广义上的多维数组，因为其可包含不同类型的元素。

## 实例

一维数组：

```r
my_array <- array(c(1:4))
print(my_array)
```

```
[1] 1 2 3 4
```

二维数组：

```r
my_array <- array(c(c(1:4), c(5:8)), dim = c(4, 4))
print(my_array)
```

```
[2,]    2    6    2    6
[3,]    3    7    3    7
```


设置各个维度的名称：

```r
vec1 <- c(5, 9, 3)
vec2 <- c(10:15)
col_names <- c("col1", "col2", "col3")
row_names <- c("row1", "row2", "row3")
mat_names <- c("mat1", "mat2")
result <- array(c(vec1, vec2), dim = c(3, 3, 2),
                dimnames = list(row_names, col_names, mat_names))
print(result)
```

```
row3    3   12   15

, , mat2

     col1 col2 col3

row2    9   11   14
row3    3   12   15
```

## 访问数组元素

类比访问坐标的方法：

```r
my_array <- array(1:12, dim = c(2, 3, 2))
print(my_array)

# 访问第1个矩阵中第1行第2列元素
result <- my_array[1, 2, 1]
print(result)

# 访问多个元素
result <- my_array[c(1, 2), c(2, 3), c(1, 2)]
print(result)

# 访问第2个矩阵中第2行元素
result <- my_array[2, , 2]
print(result)

# 访问第2个矩阵
result <- my_array[, , 2]
print(result)
```

## 操作数组元素

可直接利用访问得来的元素进行下一步计算：

```r
my_array <- array(0:9, dim = c(3, 3, 2))
print(my_array)

mat1 <- my_array[, , 1]
print(mat1)
mat2 <- my_array[, , 2]
print(mat2)

print(mat1 + mat2)
```

可以使用`apply()`对数组元素进行跨纬度计算。

语法格式：

```r
apply(X, MARGIN, FUN, ...)
```

- `X`：待使用的元素；
- `MARGIN`：应用函数的维度；
- `FUN`：要应用的函数；
- `...`：可选参数，传递给函数的其它参数。

```r
my_array <- array(1:27, dim = c(3, 3, 3))
print(my_array)

# 按层维度，对每个矩阵的行与列求和
result <- apply(my_array, 3, sum)
print(result)

# 按行维度，对每个矩阵的列与层求和
result <- apply(my_array, 1, sum)
print(result)

# 按行与列维度，对每个矩阵的层求和
result <- apply(my_array, c(1, 2), sum)
print(result)

# 按层维度，每个矩阵的第1行元素总和
result <- apply(my_array, 3, function(x) sum(x[1, ]))
print(result)
```
