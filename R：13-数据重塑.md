## 一、数据框合并：merge() 函数

`merge()` 是R基础包自带的合并函数，支持**内连接、全连接、左连接、右连接**，和数据库查询逻辑一致。

### 1. 核心参数

| 参数 | 含义 |
|------|------|
| `x, y` | 待合并的两个数据框 |
| `by` | 合并依据的**公共列名** |
| `all=TRUE` | 全连接（保留所有行） |
| `all.x=TRUE` | 左连接（保留x所有行） |
| `all.y=TRUE` | 右连接（保留y所有行） |
| `sort=TRUE` | 默认对合并列排序 |

### 2. 完整示例代码

```r
# 1. 创建两个测试数据框
table1 <- data.frame(
  SiteId = 1:6,
  Site = c("Google", "Runoob", "Taobao", "Facebook", "Zhihu", "Weibo")
)

table2 <- data.frame(
  SiteId = c(2, 4, 6, 7, 8),
  Country = c("CN", "USA", "CN", "USA", "IN")
)

# 2. 四种合并方式
# 内连接：只保留两个表匹配的行
inner <- merge(table1, table2, by = "SiteId")

# 全连接：保留两个表所有行，无匹配值为NA
full <- merge(table1, table2, by = "SiteId", all = TRUE)

# 左连接：保留左表(table1)所有行
left <- merge(table1, table2, by = "SiteId", all.x = TRUE)

# 右连接：保留右表(table2)所有行
right <- merge(table1, table2, by = "SiteId", all.y = TRUE)
```

### 3. 合并结果速记

- 内连接：**交集**
- 全连接：**并集**
- 左连接：**左表全保留，右表匹配填充**
- 右连接：**右表全保留，左表匹配填充**

## 二、数据重塑：reshape2 包

核心两个函数：

- `melt()`：**宽数据 → 长数据**（拆分）
- `dcast()`：**长数据 → 宽数据**（还原/透视表）

## 三、宽转长：melt() 函数

### 1. 用法

`melt(数据框, id = 固定不变的列)`

- `id`：指定**标识列**（不参与转换的列）
- 其余列会自动转为 `variable`（列名）+ `value`（值）

### 2. 示例代码

```r
# 创建宽格式数据
mydata <- data.frame(
  id = c(1,1,2,2),
  time = c(1,2,1,2),
  x1 = c(5,3,6,2),
  x2 = c(6,5,1,4)
)

# 宽转长：id、time保持不变，x1/x2转为变量+值
long_data <- melt(mydata, id = c("id", "time"))
```

### 转换效果

- 宽数据：一行多列指标
- 长数据：**一行一个指标值**

## 四、长转宽：dcast() 函数（数据透视表）

### 1. 核心用法

`dcast(长数据, 行 ~ 列, 聚合函数)`
- 公式格式：`行变量 ~ 列变量`
- 可选聚合函数：`mean/sum/count/length` 等

### 2. 常用透视示例

```r
# 1. 按id分组，求x1/x2平均值
dcast(long_data, id ~ variable, mean)

# 2. 按time分组，求x1/x2平均值
dcast(long_data, time ~ variable, mean)

# 3. 还原为原始宽数据（无聚合）
dcast(long_data, id + time ~ variable)

# 4. 行：id+变量，列：time
dcast(long_data, id + variable ~ time)

# 5. 行：id，列：变量+time（组合列）
dcast(long_data, id ~ variable + time)
```
