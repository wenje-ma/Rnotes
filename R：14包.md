# R：14包

## 一、查看 R 包相关信息

### 1. 查看 R 包安装路径

区分**用户自定义包**和**R 自带基础包**

```r
# 查看包的安装目录
.libPaths()
```

- 第一个路径：你自己安装的扩展包
- 第二个路径：R 预装的基础核心包

### 2. 查看电脑上**已安装**的所有包

```r
# 方式1：弹出窗口查看（可视化）
library()

# 方式2：控制台直接输出列表
installed.packages()[,1]
```

### 3. 查看当前环境**已加载**的包

```r
# 查看已载入内存、可直接使用的包
search()
```

## 二、安装 R 包

```r
install.packages("XML", repos = "https://mirrors.ustc.edu.cn/CRAN/")
```

## 三、加载 / 卸载包

### 1. 加载包

⚠️ 重启 R 后必须**重新加载**

```r
library(XML)
```

### 2. 更新包

```r
update.packages()
```

### 3. 卸载包

```r
remove.packages("XML")
```
