# R：05-循环语句

R 语言提供 **3 种循环结构** + **2 种循环控制语句**，用于重复执行代码逻辑。

## 一、循环类型

### 1. repeat 循环

**特点**：无条件直接无限执行，必须用 `break` 手动退出，否则死循环。

**适用**：不确定循环次数，需要手动终止。

#### 语法

```r
repeat {
  # 循环执行的代码
  if (退出条件) {
    break  # 满足条件则退出循环
  }
}
```

#### 标准示例

```r
cnt <- 2  # 计数器
v <- c("Google", "Runoob")

repeat {
  print(v)
  cnt <- cnt + 1
  # 满足条件退出
  if (cnt > 5) {
    break
  }
}
```

**输出**：

```
[1] "Google" "Runoob"
[1] "Google" "Runoob"
[1] "Google" "Runoob"
[1] "Google" "Runoob"
```

**执行次数**：共执行 4 次。

### 2. while 循环

**特点**：先判断条件，`TRUE` 才执行循环体；`FALSE` 直接跳过。

**适用**：不确定循环次数，但有明确终止条件。

#### 语法

```r
while (条件) {
  # 条件为 TRUE 时执行
}
```

#### 示例

```r
cnt <- 2
v <- c("Google", "Runoob")

# 条件为 TRUE 时循环
while (cnt <= 5) {
  print(v)
  cnt <- cnt + 1
}
```

**输出**：

```
[1] "Google" "Runoob"
[1] "Google" "Runoob"
[1] "Google" "Runoob"
[1] "Google" "Runoob"
```

#### 执行规则

1. 先判断条件
2. 条件为真 → 执行代码
3. 条件为假 → 结束循环

### 3. for 循环

**特点**：遍历向量/列表中的每一个元素，自动结束，无需手动计数。

**适用**：**确定循环次数**，遍历数据。

#### 语法

```r
for (变量 in 向量/序列) {
  # 循环体
}
```

#### 示例

```r
# 遍历 A B C D
v <- LETTERS[1:4]
for (i in v) {
  print(i)
}
```

**输出**：

```
[1] "A"
[1] "B"
[1] "C"
[1] "D"
```

#### 常用写法

```r
# 循环 1~5
for (i in 1:5) {
  print(i)
}
```

## 二、循环控制语句

### 1. break（终止整个循环）

**作用**：立即**结束当前所有循环**，跳出循环体继续执行后面代码。

**搭配**：最常用在 `repeat` 中，也可用于 `while/for`。

#### 示例

```r
cnt <- 2
v <- c("Google", "Runoob")
repeat {
  print(v)
  cnt <- cnt + 1
  if (cnt > 5) {
    break  # 直接退出循环
  }
}
```

### 2. next（跳过当前一次循环）

**作用**：跳过本次循环剩余代码，**直接进入下一次循环**。

**类似其他语言**：`continue`。

#### 示例

```r
v <- LETTERS[1:6]  # A B C D E F
for (i in v) {
  # 遇到 D 跳过
  if (i == "D") {
    next
  }
  print(i)
}
```

**输出**：

```
[1] "A"
[1] "B"
[1] "C"
[1] "E"
[1] "F"
```

## 三、循环对比总结表

| 循环 | 执行方式 | 退出条件 | 使用场景 |
|------|----------|----------|----------|
| `repeat` | 先执行，后判断 | 必须用 `break` | 未知次数，强制循环 |
| `while` | 先判断，后执行 | 条件为 `FALSE` | 未知次数，有条件 |
| `for` | 遍历元素自动执行 | 遍历完成自动结束 | 已知次数，遍历数据 |

## 四、核心知识点

1. **repeat 必须配合 break**，否则死循环
2. **while 循环条件一开始为 FALSE，则一次都不执行**
3. **for 循环最常用**，专门用来遍历向量、数据框
4. **break = 终止整个循环**
5. **next = 跳过本次循环**
6. `LETTERS`：R 内置大写字母向量（A-Z）
