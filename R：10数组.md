# R：10数组

数组是**同一数据类型元素**的多维集合，是 R 语言核心数据结构之一：

- 一维数组 = 向量
- 二维数组 = 矩阵
- 三维及以上 = 高维数组

列表是**广义数组**，可存储不同类型元素；数组仅支持单一类型元素。

## 一、核心知识点

1. **创建函数**：`array()`
2. **语法**

```r
array(data = NA, dim = length(data), dimnames = NULL)
```

3. **参数说明**
   - `data`：数组元素，不足时会循环填充
   - `dim`：维数向量，格式 `c(行数, 列数, 层数,...)`
   - `dimnames`：各维度名称（列表格式，可选）
4. **索引规则**：R 中数组索引**从 1 开始**

## 二、创建数组（1/2/3维 + 命名）

### 1. 一维数组（向量）

```r
# 直接传入元素，默认1维
arr1 <- array(1:4)
print(arr1)
# 输出：[1] 1 2 3 4
```

### 2. 二维数组（矩阵）

```r
# dim = c(行, 列)
arr2 <- array(1:8, dim = c(4, 4))
print(arr2)
```

```
     [,1] [,2] [,3] [,4]
[1,]    1    5    1    5
[2,]    2    6    2    6
[3,]    3    7    3    7
[4,]    4    8    4    8
```

### 3. 三维数组

```r
# dim = c(行, 列, 层)
arr3 <- array(1:8, dim = c(4, 4, 2))
print(arr3)
```

```
, , 1  # 第一层矩阵
     [,1] [,2] [,3] [,4]
[1,]    1    5    1    5
[2,]    2    6    2    6
...

, , 2  # 第二层矩阵
...
```

### 4. 带维度名称的数组

```r
# 定义各维度名称
row_names <- c("行1", "行2", "行3")
col_names <- c("列1", "列2", "列3")
mat_names <- c("矩阵1", "矩阵2")

# 创建带名称的3维数组
arr_named <- array(
  c(5,9,3, 10:15), 
  dim = c(3, 3, 2),
  dimnames = list(row_names, col_names, mat_names)
)
print(arr_named)
```

## 三、访问数组元素

格式：`数组[行索引, 列索引, 层索引]`

- 留空：表示选取该维度**所有元素**

```r
# 创建 2行3列2层 数组
arr <- array(1:12, dim = c(2, 3, 2))

# 1. 单个元素：第1层，第1行，第2列
arr[1, 2, 1]  # 输出 3

# 2. 多个元素：指定行+列+层
arr[c(1,2), c(2,3), c(1,2)]

# 3. 整行：第2层，第2行所有列
arr[2, , 2]  # 输出 8 10 12

# 4. 整个矩阵：第2层所有行、列
arr[, , 2]
```

## 四、数组元素运算

直接提取子数组/矩阵后，可做加减乘除等数学运算：

```r
arr <- array(0:9, dim = c(3, 3, 2))

# 提取两个矩阵
mat1 <- arr[, , 1]
mat2 <- arr[, , 2]

# 矩阵相加
mat1 + mat2
```

## 五、跨维度计算：apply() 函数

`apply()` 是数组**批量计算**的核心函数，支持按指定维度汇总。

### 语法

```r
apply(X, MARGIN, FUN, ...)
```

- `X`：数组对象
- `MARGIN`：计算维度（1=行，2=列，3=层，`c(1,2)`=行+列）
- `FUN`：应用函数（sum/mean/自定义函数）

### 示例

```r
# 3行3列3层 数组
arr <- array(1:27, dim = c(3, 3, 3))

# 1. 按层求和（每层矩阵的总和）
apply(arr, 3, sum)

# 2. 按行求和（所有层、列，每行总和）
apply(arr, 1, sum)

# 3. 按行+列求和（每层对应位置总和）
apply(arr, c(1,2), sum)

# 4. 自定义函数：每层矩阵的第一行求和
apply(arr, 3, function(x) sum(x[1, ]))
```
