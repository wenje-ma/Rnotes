# R：06函数

函数是**执行特定任务**的代码块，可重复调用，是 R 语言的核心。

## 一、函数基础结构

R 语言使用 `function` 关键字创建函数。

### 1. 完整语法

```r
函数名 <- function(参数1, 参数2, ...) {
  函数体（执行逻辑）
}
```

### 2. 三要素

- **函数名**：调用时使用
- **参数**：输入数据（可选）
- **函数体**：执行代码

## 二、自定义函数

### 1. 基础函数

```r
# 定义加法函数
add_numbers <- function(x, y) {
  x + y  # 自动作为返回值
}

# 调用
sum_result <- add_numbers(3, 4)
print(sum_result)
```

**输出**：`7`

### 2. 无参数函数

```r
new_function <- function() {
  for (i in 1:5) {
    print(i^2)
  }
}

new_function()  # 直接调用
```

### 3. 带参数的函数

R 支持**按位置传参** + **按名称传参**。

```r
new_function <- function(a, b, c) {
  print(a * b + c)
}

# 按位置
new_function(5, 3, 11)

# 按名称（顺序任意）
new_function(c = 11, a = 5, b = 3)
```

**输出**：`26`

### 4. 带默认值的函数

调用时不传参 → 自动使用默认值。

```r
new_function <- function(a = 3, b = 6) {
  print(a * b)
}

new_function()      # 使用默认值 3*6 → 18
new_function(9, 5)  # 覆盖默认值 → 45
```

## 三、R 函数特性：懒惰计算

**定义**：参数**只在使用时才计算**，未使用则不检查。

### 示例

```r
new_function <- function(a, b) {
  print(a)
  # print(b)  # 如果不调用 b，则不会报错
}

new_function(6)
```

- 只传 `a`，**没用到 `b`** → 不报错
- 用到 `b` 但没传 → 报错：缺少参数

## 四、函数返回值

1. **自动返回**：函数**最后一行表达式**
2. **手动返回**：可使用 `return(值)` 强制退出并返回

```r
fun <- function(x) {
  if (x < 0) {
    return("负数")
  }
  return("非负数")
}
```

## 五、常用内置函数

| 函数 | 功能 |
|------|------|
| `seq(from, to)` | 生成序列 |
| `mean(x)` | 平均值 |
| `sum(x)` | 求和 |
| `length(x)` | 长度/元素个数 |
| `paste(a, b)` | 字符串拼接 |
| `str(x)` | 查看数据结构 |
| `matrix(data, nrow)` | 创建矩阵 |

### 示例

```r
print(seq(32, 44))
print(mean(25:82))
print(sum(41:68))

x <- c(1:5)
print(sum(x))
print(mean(x))
print(length(x))
print(paste("Hello", "World"))
str(x)
```

## 六、函数知识点总结

1. **创建**：`函数名 <- function(参数) {}`
2. **调用**：`函数名(参数)`
3. **参数**：可无参、可默认、可按名传参
4. **返回**：默认返回最后一行，可用 `return()`
5. **懒惰计算**：不用的参数不检查
6. **内置函数**：直接调用，无需定义
