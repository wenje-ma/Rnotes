# R：12数据框

数据框是 R 里**最常用的表格型数据结构**，相当于 Excel 表格。

## 一、数据框特点

- 二维表格结构（行 + 列）
- **每一列是一个变量**，列名唯一
- **同一列数据类型必须相同**
- **不同列数据类型可以不同**（字符、数值、逻辑等）
- 每一行是一条完整记录

## 二、创建数据框

使用 `data.frame()` 创建。

```r
# 创建员工信息数据框
df <- data.frame(
  姓名 = c("张三", "李四"),
  工号 = c("001", "002"),
  月薪 = c(1000, 2000)
)

df
```

## 三、查看数据框

### 1. `print()`

展示**完整表格内容**。

### 2. `str()`

展示**结构 + 数据类型**。

```r
str(df)
```

### 3. `summary()`

展示**数据概要统计**。

- 字符列：长度、类型
- 数值列：最小值、分位数、中位数、均值、最大值

## 四、提取行和列

### 格式：`数据框[行, 列]`

```r
# 取第1、3列
df[, c(1, 3)]

# 取前2行
df[1:2, ]

# 取指定列
df$姓名
df$月薪
```

## 五、扩展数据框

### 1. 添加新列

```r
df$部门 <- c("运营", "技术")
```

### 2. 按列合并（cbind = column bind）

把多个向量拼成数据框/矩阵。

```r
cbind(姓名, 工号, 月薪)
```

### 3. 按行合并（rbind = row bind）

把**结构相同**的两个数据框上下拼接。
```r
rbind(df1, df2)
```
