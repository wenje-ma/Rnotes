# R：07-字符串

## 一、字符串基础

- R 字符串用 **双引号 `" "`** 定义
- 双引号内可直接包含 **单引号**
- 字符串本质是**字符向量**

```r
print("双引号字符串中可以包含'单引号'")
```

```
[1] "双引号字符串中可以包含'单引号'"
```

## 二、字符串常用操作函数

### 1. paste() —— 字符串拼接

最常用的字符串连接函数。

**语法**

```r
paste(..., sep = " ", collapse = NULL)
```

- `sep`：多个元素之间的分隔符（默认空格）
- `collapse`：将整个结果合并成一个字符串，并用指定符号连接

**示例**

```r
# 默认用空格连接
print(paste(letters[1:6], 1:6))

# 无空格分隔，并用 = 合并成一个字符串
print(paste(letters[1:6], 1:6, sep = "", collapse = "="))
```

```
[1] "a 1" "b 2" "c 3" "d 4" "e 5" "f 6"
[1] "a1=b2=c3=d4=e5=f6"
```

### 2. format() —— 格式化字符串

用于数字格式化、对齐、指定位数、科学计数法等。

**常用参数**

- `digits`：有效数字位数
- `nsmall`：小数点后最少位数
- `width`：输出宽度
- `justify`：对齐方式（left/right/centre）
- `scientific`：是否使用科学计数法

**示例**

```r
print(format(23.123456789, digits = 9))       # 指定位数
print(format(c(6, 13.14521), scientific = TRUE)) # 科学计数法
print(format(23.47, nsmall = 5))             # 小数点后补0
print(format(13.7, width = 6))               # 宽度对齐
print(format("Runoob", width = 9, justify = "l")) # 左对齐
```

### 3. nchar() —— 统计字符个数

**语法**

```r
nchar(x)
```

**示例**

```r
print(nchar("Google Runoob Taobao"))  # 统计字符串长度
print(nchar(c(1, 22, 333, 4444)))      # 向量每个元素长度
```

```
[1] 20
[1] 1 2 3 4
```

### 4. toupper() / tolower() —— 大小写转换

```r
print(toupper("Runoob"))  # 转大写
print(tolower("Runoob"))  # 转小写
```

```
[1] "RUNOOB"
[1] "runoob"
```

### 5. substring() —— 截取字符串

**语法**

```r
substring(字符串, 起始位置, 结束位置)
```

位置从 **1 开始计数**。

**示例**

```r
print(substring("Runoob", 2, 5))
```

```
[1] "unoo"
```

### 6. gsub() —— 全局字符串替换

**语法**

```r
gsub(要替换的内容, 替换成什么, 原字符串)
```

**示例**

```r
print(gsub("World", "R", "Hello, World!"))
```

```
[1] "Hello, R!"
```

### 7. strsplit() —— 字符串拆分

**语法**

```r
strsplit(字符串, 分隔符)
```

**示例**

```r
print(strsplit("Hello, World!", ","))
```

```
[[1]]
[1] "Hello"   " World!"
```
