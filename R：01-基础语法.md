# R：01-基础语法

## 一、第一个R程序：Hello World

```r
# 赋值：创建字符串变量
mystring <- "Hello World!"
# 打印输出
print(mystring)
```

**输出结果**

```
[1] "Hello World!"
```

### 编码规范

所有**中缀运算符**（`<-`、`+`、`-`、`=`等）**左右必须加空格**。

## 二、变量

### 1. 命名规范

- 统一使用 **snake_case** 命名法
- 全小写字母
- 单词之间用**下划线 `_`** 分隔
- 禁止使用：大写字母、空格、连字符、多余点号

✅ 正确：`user_name`、`data_list`
❌ 错误：`UserName`、`user-name`、`user name`

### 2. 变量赋值

**赋值符号**：优先使用 `<-`，全局赋值用 `<<-`

```r
# 赋值数字向量
var1 <- c(0, 1, 2, 3)
# 全局赋值
var2 <<- c("learn", "R")
# 混合类型（自动转换）
var3 <- c(TRUE, 1)
```

### 3. 核心函数：`c()`

`c()` = combine，作用是**将多个值打包成向量**（R最基础的数据结构）。

### 编码规范

**逗号后面必须加空格**：`c(1, 2, 3)` ✅

## 三、变量管理

### 1. 查看所有变量：`ls()`

```r
var1 <- c(0,1,2)
var2 <- c("a","b")
# 列出当前环境所有变量名
print(ls())
```

### 2. 删除变量：`rm()`

仅删除代码不会删除内存中的变量，必须用 `rm()` 手动删除

```r
# 删除单个变量
rm(var3)
# 删除多个变量
rm(var1, var2)
# 删除所有变量
rm(list = ls())
```

### 3. 函数帮助查询

```r
# 查询函数用法
?print
?ls
```

## 四、输入与输出

### 1. 标准输出：`print()`

可输出字符串、数字、科学计数法等任意R对象

```r
print("RUNOOB")
print(123)
print(3e2)  # 科学计数法 = 3*10^2
```

**特点**：输出自带 `[1]` 标号，标识这是向量的第1个元素。

### 2. 拼接输出：`cat()`

`cat()` = concatenate，用于**拼接文本并直接打印**

```r
cat(1, "加", 1, "等于", 2, "\n")
```

**输出**：`1 加 1 等于 2`

**特点**：无`[1]`标号，仅输出纯文本，支持换行符`\n`。

### 编码规范

字符串**统一使用双引号 `"`**，不推荐单引号 `'`。

## 五、文件读写

### 1. `cat()` 写入文件

- 默认：**覆盖写入**
- 追加写入：添加 `append = TRUE`

```r
# 覆盖写入文件
cat("RUNOOB", file = "1.txt")
# 追加写入
cat("GOOGLE", file = "1.txt", append = TRUE)
```

### 2. `sink()` 重定向控制台输出

将**后续所有控制台输出**自动写入文件：

```r
# 仅写入文件，控制台不输出（覆盖模式）
sink("1.txt")

# 同时写入文件 + 控制台显示（追加/换行模式）
sink("1.txt", split = TRUE)

# 关闭重定向，恢复正常控制台输出
sink()
```

### 3. 读取文件文字：`readLines()`

将文件内容读取为字符串向量

```r
txt <- readLines("1.txt")
print(txt)
```

## 六、工作目录

工作目录 = R默认读取/写入文件的路径

### 1. 查看当前工作目录：`getwd()`

```r
getwd()
```

### 2. 设置工作目录：`setwd()`

```r
setwd("你的文件夹路径")
```

### 路径规则

1. 路径分隔符用 `/`，不要用 `\`
2. 必须是**已存在的文件夹**
3. 设置后：关闭文件不失效，关闭编辑器会失效

### 路径简化效果

- 绝对路径：`readLines("C:/a/b/1.txt")`
- 设置工作目录后：`readLines("1.txt")`

## 七、R注释

R**只支持单行注释**，用 `#` 开头：

```r
# 这是单行注释
x <- 10  # 行尾注释
```

## 八、通用编码规范总结

1. 中缀运算符（`<-`、`+`等）两侧加空格
2. 逗号后必须加空格
3. 变量名用 `snake_case`
4. 字符串用双引号 `"`
5. 一行代码不超过80字符
6. 删除行尾多余空格
7. 缩进用4个空格
